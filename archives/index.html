localStorage的说明

08 Jul 2017

HTML5 提供了两种在客户端存储数据的新方法

  1. localStorage:没有时间限制的数据存储
  2. sessionStorage:针对一个session的数据存储,当用户关闭浏览器窗口后，数据会被删除

之前这些都是cookie完成的,但是cookie不适合大量数据的存储
而且在html5中,数据只有在请求时使用数据,它使在不影响性能的情况下存储大量数据成为可能

localStorage本身带有方法有
　　添加键值对：localStorage.setItem(key,value)
　　获取键值：localStorage.getItem(key)
　　删除键值对：localStorage.removeItem(key)
　　清除所有键值对：localStorage.clear()

node NPM Gulp的简单介绍

07 Jul 2017

检查是否按照node

在cmd下输入node -v 会出现node的版本
输入npm -v 会出现npm的版本

js能在浏览器中运行,是因为浏览器中有js运行的环境(例如:google chrome v8引擎)
链接查看几种js引擎的介绍

Node是一个服务器端的js运行环境,还是当做web服务器来使用
直接在cmd下敲node,然后回车,可以和浏览器的控制台很像,可以在里面执行js代码

npm

Node Package Manager node的包管理工具,分为:

1. 资源包 jquery ..        --save (-S)
2. 工具包 iting5_top..     -g 全局安装-global 全局安装,可以在整个电脑上使用

–savedev (-D) 开发过程中需要这个包,项目上线后,在生产环境不需要这个包
npm init 初始化,会生成package.json文件
npm init -y 无需填写信息

npm install jquery –save

执行完后会自动在package.json 里面生成一个依赖性,内容为已经下载的文件和版本(清单文件,当前项目需要用到的包)
会生成一个node_modules文件夹,里面放下载的文件

另外,一般给别人传送文件的时候,不把node_modules发给别人,别人收到文件后,直接在当前文件夹下输入npm install 会自动把package.json里面的依赖性下载下来,并且放在node_modules文件夹内

npm下载文件可以指定版本号
查看版本号 npm info bootstrap
指定版本号下载 npm install bootstrap@3.3.0

在cmd中内容太多,无法显示完全的问题:控制台的缓存是有限的,它只能保存指定行数的信息
方法一:在输入的原命令的基础上加上|more
例如:npm info bootstrap|more
方法二:原命令>文本文件名.txt
例如:npm info bootstrap>1.txt

搜索包
npm search 包名

移除包 npm remove [包名] –save
删除包文件,并且修改package.json的内容

使用淘宝镜像
npm install -g cnpm –registry=https://registry.npm.taobao.org

Gulp :前端自动化构建工具
例如 :压缩代码,less转为css

安装Gulp npm install gulp-cli -g
然后执行 npm install gulp -D (-D 是–savedev的缩写)
touch gulpfile.js 在当前目录下创建一个文件

Gulp应用场景
在项目上线之前,对项目进行构建,自动完成一系列重复的操作

gulp核心方法

* task() 定义一个任务
* src()  根据目录找文件 
* dest() 文件转储的地址
* watch()
  1. 新建一个文件夹,首先进行npm init -y 初始化npm,生成package.json
  2. 然后执行 npm install gulp -D (一个文件夹内需要执行这一个,把gulp中所需要的包下载下来)
  3. touch gulpfile.js
    //gulpfile.js (gulp需要做的事情需要定义到这里面)
  4. 另外新建一个js文件,里面写入部分代码(压缩代码的时候,要压缩这个文件 index.js)

var gulp=require(“gulp”);//nodejs当中模块化的实现,加载gulp模块
var uglify=require(“gulp-uglify”);

gulp.task(“js”,function(){//参数为任务的名称,例如js代码的压缩
gulp.src(“./index.js”)//根据目录找文件
//压缩代码需要插件 gulp-uglify压缩代码的插件 在cmd命令行输入npm install gulp-uglify -D 然后引入gulp-uglify 模块,在上面引入 不知道如何引,查看官网npmjs.com 输入框搜索 gulp-uglify
.pipe(dest(“./dest”)) //gulp的pipe方法(管道),把文件存储到dest文件夹下
});

  1. 在当前文件夹的cmd下执行 gulp js 命令
