博学谷项目

06 Jul 2017
  1. 退出功能,要向后台发送退出请求,后台会自动清空$.cookie(“PHPSESSID”),然后再请求成功的时候,跳转页面到登录页面(location.href)
  2. 当不是在登录页的时候,判断用户是否登录($.cookie(“PHPSESSID”)),没有登录的话,页面跳转到登录页面,强制用户进行登录
  1. 不能直接给a链接添加点击事件,如果要添加的话,最后需要return false
  2. 共用侧边栏的一些事件 require([“static/js/common”])

  3. 侧边栏注意事项

    • 给侧边栏中的li注册点击事件的时候(当前点击的背景色改变),发现页面有跳转,新增加的背景色会恢复
    • 改的是当前页面的内容,也要改变其他页面有侧边栏的内容
    • 在哪个页面的时候,让这个页面对应的导航栏的背景色改变
    • 获取location.href的值,遍历所有的a进行比较,如果a的href的值和地址值相等,就让当前的a的父元素li的背景色改变

  4. bootstrap不支持AMD,所以需要在配置文件中添加shim,并且配置bootstrap的依赖项

  5. 模板引擎中的判断语句

  6. 模板引擎中的内容要写在大括号里面

  7. 在注销启用时间的时候,思维限定,只想去重新渲染数据,其实还有更简单的方法,直接添加删除相关,更改相应的文字

  8. 把获取url中后面拼接的数据的这一部分js放在公共的js的文件中,为了使后面的模块中,可以使用它,使js中返回整个函数,在接收的时候,接收的是一个函数geturl:function ()
    若要调用函数中的返回值,在模板中定义变量util接收整个函数,然后通过util.geturl()来调用这个函数

  9. 表单提交事件注意事项

    • 使用jquery.form插件进行提交,通过form标签上的id发送ajaxSubmit请求数据 $(#id).ajaxSubmit(function())
    • 函数里面写和ajax请求一样的内容 url type success:function 注意发送的数据data
    • 给form中的表单标签添加name属性,不然ajaxform提交不上去,会自动提交数据 name=””中的值要符合data中需要传参的名称
    • 注意select 中添加name属性, option中一定要有value值,因为最终提交拿到的数据是选中的那一项的option中的value
    • input中单选的value值也一定要有

  10. 知识点 NProgress插件的使用

  11. jquery ajax全局事件
    当页面中任何一个请求被发送,ajax全局事件就会被触发 ajax所有的全局事件都有绑定到document上

  12. 表单验证插件jquery-validate

    • data-required 必填项
    • data-trim 去掉空格
    • data-pattern 设置正则表达式验证
    • data-description=”name”,name为description对象中的key值
    • data-describedby=”需要关联内容显示的id值”,
    • data-conditional=”isBeauty” 和conditional对象配合使用,isBeauty是一个函数
      在验证的时候调用这个函数,当函数的返回值为true,则验证通过

  13. 通过jquery-form插件来实现表单提交事件

    • 通过$(“.settings”)来找form是为了事件委托
    • ‘submit’ 只要表单内有button按钮,通过点击,就可以实现提交
    • $(“this”)指要提交的表单
    • ajaxSubmit()指jquery-form插件中的方法,用来提交表单
    • 提交表单的data数据为,表单中带有name属性的内容

  1. form表单提交事件注意事项(jquery.form插件)

    • 发送ajaxSubmit提交数据时,不需要再手动填写要发送的数据
    • 需要在表单标签里面填写name属性,name属性的值和接口文档中需要发送的数据名称是一样的,值为value当中的值(表单会自动获取,无需我们填写)
    • 对于select中的注意事项

  2. 局部渲染数据的方法
    重新发送ajax请求渲染数据,针对本项目中的模板中的部分进行局部渲染

  3. 判断当前点击的按钮是保存按钮还是编辑按钮
    思路如下:

    • 在模板中给button一个自定义的type属性
    • 在请求返回的数据data中,添加data.type,通过添加按钮进来的值为add,通过编辑按钮进来的type的值为edit
    • 给模板中的data.type赋值,值为返回的data.type值
    • 在点击当前按钮的时候,判断type的值,如果是add就是保存按钮,反之就是编辑按钮
